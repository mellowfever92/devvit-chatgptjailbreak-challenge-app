# ChatGPT Jailbreak Challenge Devvit App

Reddit Devvit application powering competitive jailbreak challenges for r/ChatGPTJailbreak. The app orchestrates OpenAI-powered verification and automated judging for community submissions, manages leaderboards, and provides moderation tooling.

## Development

Install dependencies and run TypeScript compilation:

```bash
npm install
npm run build
```

Configure the `openaiApiKey` app setting before deploying.
